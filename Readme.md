Simple Web Crawler
====================

This is a simple web crawler, forked from Alan Skorkin's post at http://www.skorks.com/2009/07/how-to-write-a-web-crawler-in-ruby/

It is aimed to be a good practice of writing a simple ruby application and building a small gem.
